# Portal Escolar

Site estático do projeto **Portal Escolar**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado). Esta versão funciona como um exemplo genérico e temático do produto, com dados fictícios e uma prévia interativa.

## Ideia

Um portal para publicar posts, artigos, atividades, avisos, feiras, votações de líderes e solicitações de materiais para projetos dos alunos.

## Exemplos temáticos

- **Post de atividade:** Aluno que faltou consulta resumo da aula, anexo e tarefa pendente.
- **Votação de líder:** Candidatos, prazo e resultado parcial ficam em um espaço organizado.
- **Pedido de materiais:** Projetos estudantis registram insumos necessários para análise.
