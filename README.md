# Portal Escolar

Site estático do projeto **Portal Escolar**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado).

## Ideia

Um portal para publicar posts, artigos, atividades, avisos, feiras, votações de líderes e solicitações de materiais para projetos dos alunos.
